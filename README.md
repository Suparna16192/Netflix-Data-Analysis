#Netflix Data Analysis Dashboard | Power BI
📌 Project Overview

This project presents an interactive Netflix Data Analysis Dashboard built using Power BI to explore trends in Netflix's content library. The dashboard provides insights into movies and TV shows based on release year, IMDb ratings, runtime, viewer engagement, and content classification.

The objective was to transform raw entertainment data into an intuitive dashboard that enables users to analyze content performance, identify trends, and compare different categories through interactive visualizations.

🎯 Objectives

Analyze Netflix Movies and TV Shows across different years.
Compare IMDb ratings across age certifications.
Track viewer engagement using IMDb votes.
Identify top-performing titles.
Study runtime and rating trends over time.
Create an interactive dashboard for easy exploration.

📂 Dataset

The dataset contains information about Netflix titles, including:

Title
Type (Movie / TV Show)
Release Year
IMDb Rating
IMDb Votes
Runtime
Age Certification

📊 Dashboard Features

The dashboard includes:

📅 Release Year Slicer for dynamic filtering
🎬 Movie / TV Show Filter
📈 IMDb Votes Trend Analysis
⭐ Average IMDb Score by Release Year
⏱️ Runtime Trend Analysis
🏆 Top Rated Movies & TV Shows
👥 Average IMDb Rating by Age Certification
Interactive cross-filtering between visuals

📈 Key Insights

Movies constitute a larger share of Netflix's content library compared to TV shows.
IMDb ratings remain relatively consistent across most release years, with noticeable fluctuations in older titles.
Viewer engagement has increased significantly for content released after the early 2000s.
TV Shows generally maintain slightly higher average IMDb ratings than Movies.
Certain age certifications consistently receive higher audience ratings, indicating stronger viewer preferences.
